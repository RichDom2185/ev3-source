ev3_runForDistance(motor, rotations, speed)

Causes the motor to rotate for a specified number of rotations at the specified speed. Note: this works by sending instructions to the motors. This will return almost immediately, without waiting for the motor to actually run for the specified distance. If you wish to wait, use ev3_pause.
Parameters:
Name Type Description
motor peripheral The motor
rotations number The number of rotations to turn, in motor-specific units
speed number The speed to run at, in motor-specific units
